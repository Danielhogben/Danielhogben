#### Hermes Skills
The largest open-source AI agent skill collection — **1.5M+ skills** across 3,600+ services and 35 languages.

`skill-forge` generates skills programmatically from a service catalog × action matrix. Handcrafted skills cover AI agent frameworks, security tools, home automation, and more.

A 3-machine LAN running local AI inference, game servers, and self-hosted services.

- **Pop!_OS workstation** — RTX 3050, Ollama, Unsloth
- **Ubuntu server** — Nextcloud, Rocket.Chat, Prometheus, RSMod
- **Raspberry Pi 4B** — Lightweight AI with TFT28 display
